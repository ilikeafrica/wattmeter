# WattMeter ⚡

A tiny macOS menu-bar power monitor. Reads `AppleSmartBattery` via IOKit and shows,
in real time: charge power, system draw, adapter wattage, battery health, temperature,
and time-to-full / time-to-empty.

맥북 메뉴바 전력 모니터. 충전 W · 시스템 소비 W · 어댑터 W · 배터리 건강도 · 온도 ·
완충/방전까지 남은 시간을 실시간으로 보여줍니다.

![icon](assets/AppIcon.icns)

## What it shows

- **Menu bar** — context-aware summary:
  - On battery: `🔋 78% ↓12.4W` (charge %, discharge draw)
  - Charging: `⚡38.2W · 14.1W` (into battery · system draw)
  - AC, not charging: `🔌9.8W` (system draw)
- **Dropdown** — battery %/state, time to full/empty, adapter watts, adapter input,
  charge/discharge watts, system draw, battery temperature, cycle count, max capacity %.

## Power model

```
adapter input = system draw + power into battery
battery net power = voltage × amperage   (+ charging, − discharging)
system draw = input − battery net power
```

On Apple Silicon the actual adapter input is read from `PowerTelemetryData.SystemPowerIn`.
On Intel Macs that telemetry is unavailable, so system-draw-on-AC is shown as `—`
(charge/discharge watts and everything else still work).

## Localization

Fully localized in **English** and **한국어** via `Resources/{en,ko}.lproj`.
The UI (and the app's display name — *WattMeter* / *전력 모니터*) follows the
system language automatically.

## Install

**Recommended** — download the notarized `WattMeter-x.y.dmg` from
[Releases](../../releases), open it, and drag WattMeter to Applications.
It runs in the menu bar (no Dock icon).

## Build from source

Xcode 15+ and [XcodeGen](https://github.com/yonaskolb/XcodeGen)
(`brew install xcodegen`).

```bash
xcodegen generate                 # project.yml → WattMeter.xcodeproj
open WattMeter.xcodeproj          # or build from Xcode directly

./build.sh                        # CLI: build + install to /Applications + run
```

For a distributable, **notarized** DMG:

```bash
# one-time: register notarization credentials
xcrun notarytool store-credentials wattmeter \
  --apple-id <APPLE_ID> --team-id 9ZL9DK8938 --password <app-specific-password>

./release.sh                      # archive → Developer ID export → DMG → notarize → staple
```

The app icon is generated by `assets/makeicon.swift` (no external assets).
