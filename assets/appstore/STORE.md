단순한 배터리 %가 아니라, 전력이 실제로 어떻게 흐르는지 보여줍니다.

## 🇺🇸 English

### Subtitle (30 chars)
```
Battery power in your menu bar
```

### Promotional Text (170 chars)
```
Your Mac's charge power, system draw, adapter wattage, battery health and temperature — live in the menu bar. No ads, no account, no tracking, fully offline.
```

### Description
```
WattMeter is a tiny, clean menu-bar app that shows your Mac's power status in real time.

It's more than a battery percentage — it shows how power actually flows:
• Input power coming from the adapter (W)
• Power charging into the battery (W)
• Power your system is drawing (W)
• Discharge and time remaining when on battery

■ Features
• Live menu-bar readout — the right numbers for charging / on-battery / AC
• Time to full and time to empty
• Adapter wattage and description, measured input power
• Battery temperature, cycle count, and maximum capacity (health)
• Automatic Korean and English
• Universal — Apple Silicon and Intel

■ Light and honest
• Runs only in the menu bar, no Dock icon
• No ads, no account, no internet required
• Collects and transmits absolutely no data

Power balance: adapter input = system draw + power into the battery.
WattMeter reads macOS battery data (IOKit) directly and computes the rest.

Note: measured input power (adapter→system) is provided on Apple Silicon Macs. On Intel Macs, charge/discharge and battery info are shown, but system draw may be unavailable.
```
